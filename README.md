# Paradise Nursery shopping application

A simple online plant shop built using React, Vite, and Redux.  
Users can browse plants, add them to a shopping cart, adjust quantities, and view the total cost.

---

## Features

- Landing page with Get Started button
- Navigation bar with Home, Products, and Cart
- Plant product listing with categories
- Add to cart functionality
- Shopping cart with:
  - Increase and decrease quantity
  - Remove items
  - Automatic removal when quantity is 0
  - Total price calculation
- Cart item count displayed in the navbar

---

## Tech Stack

- React
- Vite
- Redux Toolkit
- CSS
